## 5. System Design Methodology

### Core Approach (First Principles)
1. Identify inputs and outputs
2. Map business process
3. Map technical process
4. Build system logic
5. Validate using Claude
6. Visualise in Excalidraw
7. Translate to Miro
8. Gut-check system integrity

### Design Principles
- First principles thinking
- Deterministic logic before AI
- Visual validation before build
